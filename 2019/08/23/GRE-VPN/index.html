一、GRE隧道VPN

1.1 扩展

L2TP-VPN ：总公司和分公司之间
IPsec-VPN：总公司和分公司之间（安全性高）
PPTP-VPN ：基于个人连接公司内部网络
IBGP：(Internal Border Gateway Protocol)——内部BGP协议(IBGP)仅用于多归属场合。IBGP允许边缘路由器共享NLRI及其相关属性，从而增强系统范围内的路由策略。此外，当位于转接（transit）AS中的边缘路由器将学习自外部对等体的路由传递给其他边缘路由器（以便将该路由宣告给外部对等体）时，就可以利用IBGP来作为传递手段。–EGP
IGP：内部网关路由协议（局域网）在同一个自治系统内交换路由信息，RIP，OSPF和IS－IS都属于IGP。IGP的主要目的是发现和计算自治域内的路由信息。

IEGP：

1.2 基本概念

GRE：（Generic Routing Encapsulation 通用路由封装协议） 创建虚拟隧道，实现隧道功能，通过隧道连接两个远程网络，如同直连。
缺点：隧道能力极强，但安全性很差。

1.3 GRE的隧道VPN配置

1.3.1 基本配置

PC1(config)#no ip routing
PC1(config)#ip default-gateway 192.168.1.1
PC1(config)#int e0/0
PC1(config-if)#ip add 192.168.1.10 255.255.255.0
PC1(config-if)#no shutdown

BJ(config)#int e0/0
BJ(config-if)#ip add 192.168.1.1 255.255.255.0
BJ(config-if)#no shutdown
BJ(config)#int e0/1
BJ(config-if)#ip add 61.1.1.1 255.255.255.0
BJ(config-if)#no shutdown

Internet(config)#int e0/1
Internet(config-if)#ip add 61.1.1.2 255.255.255.0
Internet(config-if)#no shutdown
Internet(config)#int e0/0
Internet(config-if)#ip add 181.1.1.1 255.255.255.0
Internet(config-if)#no shutdown

CD(config)#int e0/0
CD(config-if)#ip add 181.1.1.2 255.255.255.0
CD(config-if)#no shutdown
CD(config)#int e0/1
CD(config-if)#ip add 172.16.1.1 255.255.255.0
CD(config-if)#no shutdown

PC2(config)#no ip routing
PC2(config)#ip default-gateway 172.16.1.1
PC2(config)#int e0/1
PC2(config-if)#ip add 172.16.1.10 255.255.255.0
PC2(config-if)#no shutdown

1.3.2 ospf动态路由配置

（不用通告PC端）

BJ(config)#router ospf 1
BJ(config-router)#network 61.1.1.1 0.0.0.0 area 0

Internet(config)#router ospf 1
Internet(config-router)#network 61.1.1.2 0.0.0.0 area 0
Internet(config-router)#network 181.1.1.1 0.0.0.0 area 0

CD(config)#router ospf 1
CD(config-router)#network 181.1.1.2 0.0.0.0 area 0

1.3.3 GRE隧道VPN配置

BJ(config)#int tunnel 0
BJ(config-if)#tunnel source 61.1.1.1
BJ(config-if)#tunnel destination 181.1.1.2
BJ(config-if)#ip address 10.0.0.2 255.255.255.252
BJ(config)#ip route 172.16.1.0 255.255.255.0 tunnel 0 //告诉去往172.16.1.0/24的vpn数据包下一跳出接口走隧道

CD(config)#int tunnel 0
CD(config-if)#tunnel source 181.1.1.2
CD(config-if)#tunnel destination 61.1.1.1
CD(config-if)#ip add 10.0.0.1 255.255.255.252
CD(config)#ip route 192.168.1.0 255.255.255.0 tunnel 0 //告诉去往192.168.1.0/24的vpn数据包下一跳出接口走隧道
