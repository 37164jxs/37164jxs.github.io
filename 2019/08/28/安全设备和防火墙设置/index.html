一、防火墙

1、基本概念

防火墙技术是通过有机结合各类用于安全管理与筛选的软件和硬件设备，帮助计算机网络于其内、外网之间构建一道相对隔绝的保护屏障，以保护用户资料与信息安全性的一种技术。

1.1 防火墙的作用
  • 控制
    在网络连接点上简历一个安全控制点，对进出数据进行限制。
  • 隔离
    将需要保护的网络与不可信任网络进行隔离，隐藏信息并进行安全防护。
  • 记录
    对进出数据进行检查，记录相关信息。
1.2 防火墙的不足
  • 作为网络边界防护机制而先天无法防范的安全威胁
    1. 来自网络内部的威胁
    2. 通过非法外联的网络攻击
    3. 计算机病毒传播
  • 由于技术瓶颈问题目前还无法有效防范的安全威胁
    1. 针对开开放服务安全漏洞的渗透攻击
    2. 针对网络客户端程序的渗透攻击
    3. 基于隐蔽通道进行通信的木马或僵尸病毒

2、防火墙配置

2.1 路由模式防火墙配置

2.1.1 所有设备的基本配置

PC1(config)#no ip routing
PC1(config)#ip default-gateway 192.168.1.1
PC1(config)#int e0/0
PC1(config-if)#ip add 192.168.1.10 255.255.255.0
PC1(config-if)#no shutdown

R1(config)#int e0/0
R1(config-if)#ip address 192.168.1.1 255.255.255.0
R1(config-if)#no shutdown
R1(config)#int e0/1
R1(config-if)#ip add 10.0.0.2 255.255.255.252
R1(config-if)#no shutdown

pixfirewall(config)# int e1
pixfirewall(config-if)# ip add 10.0.0.1 255.255.255.252
pixfirewall(config-if)# no shutdown
pixfirewall(config)# int e0
pixfirewall(config-if)# ip add 61.1.1.1 255.255.255.0
pixfirewall(config-if)# no shutdown

R2(config)#int e0/0
R2(config-if)#ip add 61.1.1.2 255.255.255.0
R2(config-if)#no shutdown

2.2 给内网路由和外网路由配置静态路由

R1(config)#ip route 61.1.1.0 255.255.255.0 10.0.0.1
R2(config)#ip route 0.0.0.0 0.0.0.0 61.1.1.1

2.3 防火墙配置

pixfirewall(config)# int e0
pixfirewall(config-if)# nameif outside
pixfirewall(config-if)# security-level 0

pixfirewall(config)# int e1
pixfirewall(config-if)# nameif inside
pixfirewall(config-if)# security-level 100
// 用防火墙划分内网，外网，配好接口ip地址以及安全级别。
pixfirewall(config)# route inside 192.168.1.0 255.255.255.0 10.0.0.2
pixfirewall(config)# route outside 0.0.0.0 0.0.0.0 61.1.1.2
// 防火墙静态路由配置
pixfirewall(config)# access-list 1 permit icmp any any
pixfirewall(config)# access-group 1 in interface outside
//应用在接口
pixfirewall(config)# nat (inside) 1 192.168.1.0 255.255.255.0
pixfirewall(config)# global (outside) 1 interface
// 防火墙上的NAT配置，实现内网访问外网。

2.4 结果

内网（PC）ping防火墙端口ping不通，但是ping外网（R2）可以ping通
外网（R2）ping内网（PC）ping不通

2.2 透明模式防火墙配置

2.2.1 基本配置

PC1(config)#no ip routing
PC1(config)#ip default-gateway 192.168.1.1
PC1(config)#int e0/0
PC1(config-if)#ip add 192.168.1.10 255.255.255.0
PC1(config-if)#no shutdown

R1(config)#int e0/0
R1(config-if)#ip add 192.168.1.1 255.255.255.0
R1(config-if)#no shutdown
R1(config)#int e0/1
R1(config-if)#ip add 10.0.0.2 255.255.255.248
R1(config-if)#no shutdown

R2(config)#int loopback 0
R2(config-if)#ip add 61.1.1.1 255.255.255.0
R2(config-if)#no shutdown
R2(config)#int e0/0
R2(config-if)#ip add 10.0.0.1 255.255.255.248
R2(config-if)#no shutdown

2.2.2 配置静态路由

R1(config)#ip route 61.1.1.0 255.255.255.0 10.0.0.1
R2(config)#ip route 192.168.1.0 255.255.255.0 10.0.0.2

2.2.3 透明防火墙配置

pixfirewall(config)# firewall transparent
//设置透明模式
pixfirewall(config)# ip add 10.0.0.3 255.255.255.248
pixfirewall(config)# int e0
pixfirewall(config-if)# nameif outside
pixfirewall(config-if)# security-level 0
pixfirewall(config-if)# no shutdown
pixfirewall(config)# int e1
pixfirewall(config-if)# nameif inside
pixfirewall(config-if)# security-level 100
pixfirewall(config-if)# no shutdown

pixfirewall(config)# access-list AA extended permit icmp any any
pixfirewall(config)# access-group AA in interface inside
pixfirewall(config)# access-group AA in interface outside
